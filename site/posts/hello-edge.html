Hello from the edge

This blog is served from the same repository that powers the GitHub profile for neko233-com.

The boundary is simple: GitHub renders README.md, while Cloudflare serves the static site in site/. The profile can stay animated and data-rich without mixing deployment notes into the account page.

Cloudflare Pages can deploy the site with no build command. Cloudflare Workers can serve the same directory through static assets using wrangler.toml.
